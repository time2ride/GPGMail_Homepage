The PGP plugin for Apple Mail.

Installation instructions

  1. You need these files

    • latest package of "GnuPG 1.x" from MacGPG (see section "GnuPG for OS X")
    • GPGMail (from downloads section)

  2. Installing the packages

    3. First install the GnuPG package and then the GPGMail package.

  3. Generating a key

    4. At the moment you need to do this in Terminal.app (Applications/Utilities/Terminal.app).
    Please see Doku on gnupg.org.
